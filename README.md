# 享樂酒店訂房平台

### Node Version
```
20.10.0
```
### Get Started
```
npm install
npm run dev
```
### production
```
npm run bulid
```
## `Git`提交規範
  - `feat` 增加新功能
  - `fix` 修復問題/BUG
  - `style` 調整風格且不影響運行結果
  - `perf` 優化/性能提升
  - `refactor` 重構
  - `revert` 撤銷修改
  - `test` 測試相關
  - `docs` 文檔/註釋
  - `chore` 配置檔修改
  - `types` 類型定義文件修改
  - `wip` 開發中尚未完成
